# AI智能文本摘要+情感分析 Web项目
## 项目信息
开发者：谢霆邦
学号：423240616
方向：自然语言处理NLP（前沿AI应用）

## 功能介绍
1. 输入长文本，AI自动生成精简摘要（支持自定义长短参数）
2. 智能判断文本情感：积极/消极，输出置信度
3. 模型可替换、参数可优化、代码可二次迁移开发

## 技术栈
- 后端：Python Flask
- AI模型：HuggingFace BART摘要模型 + DistilBERT情感模型
- 前端：HTML+JS原生开发

## 运行步骤
1. 安装依赖：pip install -r requirements.txt
2. 运行程序：python app.py
3. 浏览器打开：http://127.0.0.1:5000
